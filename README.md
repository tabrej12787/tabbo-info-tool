🔎 TABBO NUMBER INFO TOOL

A professional OSINT mobile number lookup tool designed for Termux / Linux environments.

- Name
- Father Name
- Address
- Network Circle
- Unique ID

- 📜 Search History

## 🚀 Installation (Termux)

### 1️⃣ Update packages
```bash
pkg update -y
```

### 2️⃣ Install dependencies
```bash
pkg install git python -y
```

### 3️⃣ Clone repository
```bash
git clone https://github.com/tabrej12787/tabbo-info-tool.git
```

### 4️⃣ Enter project folder
```bash
cd tabbo-info-tool
```

### 5️⃣ Install Python libraries
```bash
pip install requests colorama
```

### 6️⃣ Create required files
```bash
echo [] > history.json
echo {} > limit.json
```

### 7️⃣ Run the tool
```bash
python tabbo.py
```
